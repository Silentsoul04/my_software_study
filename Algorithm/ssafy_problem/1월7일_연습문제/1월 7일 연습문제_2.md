## 모음 제거하기

> 다음 문장의 모음을 제거하여 출력하세요.

입력 : "Life is too short, you need python"

출력 : Lf s t shrt, y nd pythn

```python
#내가 푼거
s = list(input(''))
mo = ['a', 'e', 'i', 'o', 'u']
for i in s[:]:
    if i in mo:
        s.remove(i)
print("".join(s))

#쌤이 푼거
a = "Life is too short, you need python"
answer= ''
for i in a:
    if i not in mo:
        answer += i
print(answer)
```
